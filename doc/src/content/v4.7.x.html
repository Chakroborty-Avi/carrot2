Carrot2 4.7.x

Release history for Carrot2 4.7.0 and bugfix releases.

Version 4.7.0

This release includes an upgrade of dependencies, build plugins and a switch to Java 17 as the minimum runtime platform.

Dependency upgrades

jackson to 2.18.2, jetty to 11.0.24, Lucene to 9.12.1, commons-text to 1.13.0, slf4j to 2.0.17, log4j to 2.23.1.
